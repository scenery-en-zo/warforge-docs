# Related Products Section

**File:** `sections/related-products.liquid`

Displays products related to current product (by type, vendor, tags).

## Features

- Auto-similar products
- Manual selection option
- Carousel layout
- Configurable count

## Settings

| Setting | Type | Description |
|---------|------|-------------|
| `heading` | String | Section title |
| `source` | Select | Auto or manual |
| `products_to_show` | Number | Product count |
| `layout` | Select | Carousel or grid |
