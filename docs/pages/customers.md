# Customer Account Pages

Customer account pages allow logged-in users to manage their profile, orders, and addresses.

## Pages

### Login (`customers/login.liquid`)
- Customer login form
- "Forgot password" link
- Guest checkout option
- Social login (if enabled)

### Register (`customers/register.liquid`)
- New customer registration
- Email and password
- Marketing opt-in
- Social registration (if enabled)

### Account (`customers/account.liquid`)
- Order history
- Account details
- Address management
- Logout link

### Order (`customers/order.liquid`)
- Order details
- Line items
- Shipping address
- Payment status
- Fulfillment status

### Addresses (`customers/addresses.liquid`)
- Address list
- Add new address
- Edit existing address
- Delete address
- Set default address

## Features

- ✅ Shopify authentication
- ✅ Password reset flow
- ✅ Order tracking
- ✅ Address book
- ✅ Account settings
