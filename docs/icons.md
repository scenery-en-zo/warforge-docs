## Component Overview

The icon component renders inline SVG icons with consistent sizing, colors, and styling. Icons are used for:

- **Navigation** — Hamburger menu, search toggle
- **Buttons** — Add to cart, wishlist, share actions
- **Forms** — Input field indicators (email, phone)
- **Notifications** — Announcements, alerts, badges
- **Product details** — Shipping info, stock status

## Available Icons

### Core Navigation

| Icon | Use Case |
|------|----------|
| `menu` | Mobile menu toggle |
| `close` | Close modal/drawer |
| `search` | Search input icon |
| `chevron-down` | Dropdown indicators |

### E-commerce Actions

| Icon | Use Case |
|------|----------|
| `cart` | Cart icon, cart badge |
| `user` | Account/login icon |
| `user-plus` | Register account icon |
| `check` | Success/checkmark |
| `close` | Error/close indicator |

### Information & Features

| Icon | Use Case |
|------|----------|
| `truck` | Free shipping announcement |
| `package` | Package/tracking info |
| `clock` | Time/availability |
| `star` | Rating/reviews |
| `share` | Social sharing |

### UI Components

| Icon | Use Case |
|------|----------|
| `plus` | Add button |
| `minus` | Remove button |
| `filter` | Filter toggle |
| `grid` | Grid view toggle |
| `list` | List view toggle |

## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `name` | String | Required | Icon name from library |
| `size` | Select | `md` | Icon size: `sm`, `md`, `lg`, `xl` |
| `variant` | Select | `primary` | Color variant: `primary`, `secondary`, `muted`, `accent`, `success`, `warning`, `error`, `info` |
| `class` | String | — | Additional CSS classes on wrapper |
| `aria_hidden` | Boolean | `true` | Hide from screen readers (decorative) |
| `aria_label` | String | — | Accessibility label for meaningful icons |

## Size Variants

| Size | Width/Height | Use Case |
|------|-------------|----------|
| `sm` | 16×16px | Small indicators, inline text |
| `md` | 20×20px | Standard buttons, inputs |
| `lg` | 24×24px | Large buttons, prominent features |
| `xl` | 32×32px | Hero sections, featured items |

## Color Variants

| Variant | Color | Use Case |
|---------|-------|----------|
| `primary` | #F97316 (Orange) | Main CTAs, important actions |
| `secondary` | #94A3B8 (Gray) | Neutral actions, secondary UI |
| `muted` | #64748B (Muted Gray) | Decorative icons, background elements |
| `accent` | #F59E0B (Gold) | Highlights, special features |
| `success` | #22C55E (Green) | Success states, in-stock |
| `warning` | #F59E0B (Orange) | Warnings, attention needed |
| `error` | #EF4444 (Red) | Errors, out of stock |
| `info` | #3B82F6 (Blue) | Informational icons |

## CSS Classes

| Class | Description |
|-------|-------------|
| `.icon` | Base icon wrapper with SVG container |
| `.icon--sm` | Small icon (16px) |
| `.icon--md` | Medium icon (20px) |
| `.icon--lg` | Large icon (24px) |
| `.icon--xl` | Extra large icon (32px) |

## Accessibility

- Decorative icons have `aria-hidden="true"` by default
- Meaningful icons include accessible labels
- Focus visible on interactive icon buttons
- Screen readers announce labeled icons correctly
