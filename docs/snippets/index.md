# Snippets

Snippets are reusable Liquid components that can be rendered across your theme using the `{% render %}` tag. They are the building blocks of the Warforge theme.

## Available Snippets

### UI Components
- [Product Cards](./snippets-product-cards) - Product display cards
- [Buttons](./snippets-buttons) - Button variants
- [Icons](./snippets-icons) - Icon library
- [Badges](./snippets-badges) - Status badges

### Form Components
- [Inputs](./snippets-inputs) - Text input fields
- [Checkboxes](./snippets-checkboxes) - Checkbox inputs
- [Textareas](./snippets-textareas) - Multi-line text fields

### Navigation
- [Breadcrumbs](./snippets-breadcrumbs) - Breadcrumb navigation
- [Pagination](./snippets-pagination) - Page navigation

### Content
- [Article Cards](./snippets-article-cards) - Blog post cards
- [Tables](./snippets-tables) - Data tables
- [Images](./snippets-images) - Responsive images

### Utilities
- [Cookie Banner](./snippets-cookie-banner) - Cookie consent

## Usage

```liquid
{% render 'snippet-name', param1: value1, param2: value2 %}
```
